# Spring Boot GraphQL Integration
___
* In this project, CRUD operations were performed with Spring boot and GraphQL integration. 
* Integration test coverage is 100% are available.

### Tech Stack
___
- _Java 17_
- _Spring Boot 3.0.6_
- _Spring Data JPA_
- _GraphQL_
- _H2 In memory database_
- _JUnit 5_
